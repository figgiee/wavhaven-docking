Submission Guidelines

Quality standards and requirements for content submission on WavHaven

Audio Quality Standards

Minimum Requirements

  • Sample Rate: 44.1kHz or higher
  • Bit Depth: 16-bit minimum, 24-bit recommended
  • Format: WAV, AIFF, or high-quality MP3 (320kbps)
  • No clipping or distortion
  • Clean mix with proper headroom

Content Types & Specifications

Beats

  • Full length: 2-4 minutes recommended
  • Maximum file size: 100MB
  • Must include BPM and key information
  • Clear song structure

Loops

  • Perfect loop points required
  • Maximum file size: 50MB
  • Common lengths: 2, 4, or 8 bars
  • Must be royalty-free

Sound Kits

  • Organized folder structure
  • Maximum size: 500MB
  • Clear labeling of all sounds
  • README file with kit information

Presets

  • Supported plugin formats only
  • Version compatibility noted
  • Demo audio required
  • Installation instructions included

Metadata Requirements

  • Descriptive title that accurately represents the content
  • Appropriate genre tags
  • Mood and style tags
  • Key musical elements noted (instruments, effects, etc.)
  • Clear description of included files and formats

Content Restrictions

  • No copyrighted material without proper licensing
  • No samples from other artists without permission
  • No offensive or inappropriate content
  • No low-quality or unfinished work
  • No repackaged or duplicate content

Cover Art Requirements

  • Minimum size: 1000x1000 pixels
  • Maximum size: 2MB
  • Formats: JPG, PNG, or WEBP
  • Professional quality and relevant to content
  • No explicit or offensive imagery
